
## Getting started


### Install Node
```sh
$ nvm install 12.7.0
$ nvm use 12.7.0
```

### Clone this repo (and `cd` to it)
```sh
$ git clone https://github.com/vadia007/bot.git
$ cd bot
```

### Install Dependencies
```sh
$ npm install
```

### Load env vars
```sh
cp .env-sample .env
```
Input the desired environment variables in `.env`.

### Start the server
```sh
$ npm run start
```
